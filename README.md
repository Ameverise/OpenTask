# OpenTask

OpenTask — учебное веб-приложение для управления задачами. Проект помогает создавать задачи, отслеживать их выполнение, редактировать данные и сохранять список после перезагрузки страницы.

## Возможности

- добавление задач с названием, описанием, категорией, приоритетом и сроком;
- просмотр полного списка задач;
- редактирование существующих задач;
- удаление задач;
- отметка задачи как выполненной;
- фильтрация по статусу, категории и приоритету;
- поиск по названию задачи;
- статистика по общему количеству, активным и выполненным задачам;
- сохранение данных в localStorage.

## Технологии

## Запуск

Установите зависимости:

```bash
npm install
```

Запустите проект в режиме разработки:

```bash
npm run dev
```

Соберите production-версию:

```bash
npm run build
```

Откройте локальный адрес, который покажет Vite после запуска команды разработки.
